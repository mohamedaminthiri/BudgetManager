Un plan de prévoyance est représenté par :
	- Un nom
	- Une validitée
	- Des charges
	- Montant de base
	- Un ID d'utilisateur

Une charge est représentée par :
	- Un nom
	- Un type (frais, gains)
	- Une somme (frais -> rouge, gain -> vert)
	- Un nombre de répétitions
	- Un/des plans de prévoyances associés
	- Un commentaire
	- Un ID de plan prévisionel
